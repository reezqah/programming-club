# Task 2 - C++ Assignment | Programming Club

##  Description
This project contains usage of the following in C++:
- Arrays
- Pointers
- References
- Structures

## Files
- array.cpp → Prints array using for loop and for-each loop
- pointer.cpp → Demonstrates pointer usage
- reference.cpp → Demonstrates reference variables
- structure.cpp → Rectangle structure and area calculation

##  How to Run

Compile and run using g++:

g++ array.cpp -o array
./array

g++ pointer.cpp -o pointer
./pointer

g++ reference.cpp -o reference
./reference

g++ structure.cpp -o structure
./structure
